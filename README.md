# 🚗 Lane Detection & Steering Prediction System

## Overview
This project detects road lanes and predicts steering direction using computer vision techniques.

## Tech Stack
- Python
- OpenCV
- NumPy

## Features
- Lane detection using Hough Transform
- Steering prediction (Left / Right / Straight)
- Noise filtering using slope and ROI

## How to Run
pip install opencv-python numpy
python LaneDetectionSystem.py
